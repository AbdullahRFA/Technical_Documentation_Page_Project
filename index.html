Introduction

Python is a high-level, interpreted programming language known for its simplicity, readability, and versatility. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its clean syntax, making it a popular choice for both beginners and experienced developers. Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.


  • Most used in Data Science
  • And Machine Learning
Getting Started

1. Windows/MacOS: Download Python from the official Python website and follow the installation instructions.

2. Linux: Most distributions come with Python pre-installed. To check, run python3 --version in the terminal. To install Python, you can use your package manager:


sudo apt-get install python3

  • Running Python Code

  • Interactive Mode: Open the terminal and type python3 to start the Python interpreter.
  • Script Mode: Save your code in a file with the .py extension and run it from the terminal with:


    • python3 your_script.py

Data Types

Python has several built-in data types:


1. Numeric Types:

  • int: Integer, e.g., 10
  • float: Floating-point number, e.g., 10.5
  • complex: Complex number, e.g., 3+4j

2. String Types: A sequence of characters, e.g., "Hello"


3. Boolean Types: Represents True or False


4. Data Structures:

  • List: Ordered, mutable collection, e.g., [1, 2, 3]
  • Tuple: Ordered, immutable collection, e.g., (1, 2, 3)
  • Set: Unordered collection of unique elements, e.g., {1, 2, 3}
  • Dictionary: Key-value pairs, e.g., {"key": "value"}
Control Flow Statement

Conditions


Use if, elif, and else for decision-making.


x = 10
if x > 0:
print("Positive")
elif x == 0:
print("Zero")
else:
print("Negative")

Loop

  • For Loop

    • Iterates over a sequence.

    for i in range(5):
    print(i)

  • While Loop

    • Repeats as long as a condition is true.

    count = 0
    while count < 5:
    print(count)
    count += 1
Functions

Functions are reusable blocks of code. Define a function with the def keyword.


def greet(name):
return f"Hello, {name}"
print(greet("Python"))

Lambda Functions

Anonymous, one-line functions defined with the lambda keyword.


add = lambda x, y: x + y
print(add(5, 3))
Object-Oriented Programming (OOP)

Python supports OOP concepts like classes and objects, inheritance, and polymorphism.


Classes and Objects


class Dog:
def __init__(self, name):
self.name = name
def bark(self):
return f"{self.name} says Woof!"
my_dog = Dog("Buddy")
print(my_dog.bark())

Inheritance


class Animal:
def speak(self):
return "Some sound"
class Cat(Animal):
def speak(self):
return "Meow"
my_cat = Cat()
print(my_cat.speak())
Modules and Packages

Organize your code into modules (Python files) and packages (folders containing modules).


Importing Modules

Error Handling

Use try and except blocks to handle exceptions gracefully.


try:
print(1 / 0)
except ZeroDivisionError:
print("Cannot divide by zero!")
